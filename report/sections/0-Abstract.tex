    Commonsense knowledge graph models like COMET-BART are typically evaluated using aggregate metrics, which may obscure meaningful variation across different relation types
    This study investigates whether COMET-BART's performance on the ATOMIC$^{2020}$ test set varies across the 23 relation types and three broad categories defined by \citet{hwang2021cometatomic}.
    We partition the existing BART evaluation outputs by relation type and category, computing BLEU-1 through BLEU-4 seperately for each subset.
    Category-level differences are small, but per-relation variance is substantial. \code{HinderedBy} and \code{ObjectUse} score highest, while \code{Causes}, \code{HasProperty}, and \code{CapableOf} score lowest, with class imbalance partly explaining the latter.
