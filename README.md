# Key Topics in Artificial Intelligence - Project (IM0712)

This repository contains the coursework for the **IM0712: Key Topics in Artificial Intelligence** module. The project's topic is '**Generation of Common Sense Knowledge**'

## Project description

An exciting field in literature is the use of knowledge graphs to learn models for common-sense reasoning. A recent paper on this topic [1], that (at the time of writing this assignment) will appear soon at a major AI conference; a pre-print of this paper can be found at <https://arxiv.org/abs/2010.05953>. The code and models that belong to this paper can be found at <https://github.com/allenai/comet-atomic-2020>. In brief, the paper describes a system where a neural network model was learned from a large knowledge graph and the authors use this neural network to generate new knowledge in natural language. For example, asking the question: *What would the effect be of choosing this assignment be for PersonX*, the system answers:

```bash
[[‘PersonX learns something new’, ‘PersonX gets the assignment’,
 ‘PersonX gets a job’, ‘learns something new’, ‘none’]]
```

[1] J. D. Hwang, C. Bhagavatula, R. Le Bras, J. Da, K. Sakaguchi, A. Bosselut,
and Y. Choi, “(comet-) atomic 2020: On symbolic and neural common
sense knowledge graphs,” Proceedings of the AAAI Conference on Artificial
Intelligence, vol. 35, pp. 6384–6392, May 2021.
